Pseudo Code crystal game
    //sort of like black jack\\
you will click on a crystal, each crystal will have a seperate value
at the start of every game a random number will generate, your goal is to essentaly play blackjack
    example
        if the random number generated is 30 then you need to select a crysltal, then determin its value and reach the random number generated 
        if you reach the number then you win
        if you go over the number then you lose
        each time you begin a game a random number will generate for your goal
        each one of the crystals will have their own random number generated
        make these random numbers for the crystals 1-10
